日频因子的多空组合（Long-Short Portfolio）收益率是量化因子研究中最常见的绩效评估方式之一，其计算方法在业界已经非常规范，以下是详细的标准做法（基本是海内外量化机构、基金、论文通用的规范）：

### 1. 多空组合收益率的基本定义
多空组合收益率 = 多头组合收益率 − 空头组合收益率  
即：做多因子值最高的一组股票，同时做空因子值最低的一组股票，赚取的是因子值从高到低方向的超额收益。

### 2. 典型的分组方式（最常见也是最规范的）
- 按因子值将全市场股票等权或流通市值加权排序
- 通常分成 5 组或 10 组（quintile 或 decile）
- 最常用的是：
    - 多头：因子值最高的一组（第10组或Top 20%）
    - 空头：因子值最低的一组（第1组或Bottom 20%）
- 也有极端做法取 Top 10% 和 Bottom 10%，或者 Top/Bottom 5%，但主流仍是10等分取两端

### 3. 日频多空组合收益率的具体计算步骤（日度）
以10等分为例，每天（每个交易日）都重新构建一次组合：

1. 在第 t 日收盘后，用第 t 日的因子值对所有股票排序并10等分
2. 多头组合：第10组所有股票，采取等权或流通市值加权
3. 空头组合：第1组所有股票，同样等权或流通市值加权
4. 计算第 t+1 日两组各自的收益率：
    - 多头收益率 Rt+1,Long = 第10组股票在 t+1 日的加权平均收益率
    - 空头收益率 Rt+1,Short = 第1组股票在 t+1 日的加权平均收益率
5. 第 t+1 日的多空组合收益率：
   Rt+1,LS = Rt+1,Long − Rt+1,Short  
   （实际操作中通常是等金额多空，所以直接相减即可）

6. 每天重复上述过程，就得到一个日频的多空收益率序列 {Rt,LS}

### 4. 计算多少日？
- 只要你的因子数据和股票收益率数据是日频的，就可以一直往回算（比如2010年1月1日至今）
- 实际回测中常见的是：
    - A股：2010年或2015年至今（取决于去极值和中性化方式）
    - 美股：更早（2000年甚至更早）
- 没有“必须算多少日”的硬性规定，但实务中至少需要5~10年以上的日度数据才具有统计意义。

### 5. 目前业界最规范的做法（2025年现状）
国内顶级量化私募（如幻方、九坤、明汯、天演、因诺等）和公募量化基金基本都遵循以下标准：

| 项目                  | 主流规范                                      |
|-----------------------|-----------------------------------------------|
| 分组数量              | 10等分（最常见），也有5等分                   |
| 多空取组              | Top 10%（第10组） vs Bottom 10%（第1组）      |
| 持仓权重              | 流通市值加权（最常见），也有等权              |
| 换仓频率              | 每日换仓（日频因子必须如此）                  |
| 交易成本              | 扣万分之五到万分之八（双边），视策略而定      |
| 是否做行业市值中性    | 必须做（常见做法：行业中性+市值中性）         |
| 是否对冲市场          | 常见做法：多空收益再减去市场收益率（即超额）  |
| 常见指标              | 年化收益率、年化超额、信息比率（IR）、最大回撤、胜率、年化换手率等 |

### 6. 经典论文与业界参考
- Fama-French 三因子、五因子论文：月频，但多空构建逻辑完全一样
- AQR、WorldQuant、AlphaLens、101 Formulaic Alphas 论文：全部是日频多空等权或市值加权
- 国内：中证800、沪深300、科创50等指数公司发布因子指数也都是按此逻辑

总结一句话：
日频因子的多空组合收益率 = 每天用当日因子值10等分，取最高组做多、最低组做空，计算第二日多头收益率减空头收益率，持续滚动，即可得到日度多空收益序列。这是目前最标准、最规范、业界最通用的做法。
